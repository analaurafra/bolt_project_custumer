# 🔧 Bolt Project Customer

A comprehensive customer management solution developed with **TypeScript**, designed to provide a robust and scalable experience.

## 📋 About the Project

**Bolt Project Customer** is a modern application for managing customer information, facilitating operations and improving business efficiency. Built with best development practices, it offers a solid foundation for expansion and customization.

## 🚀 Key Features

- ✅ Complete customer management
- ✅ Intuitive and responsive interface
- ✅ Scalable and modular architecture
- ✅ Type-safe with TypeScript
- ✅ Well-organized and maintainable code

## 🛠️ Technologies Used

- **Language**: TypeScript

## 📦 Installation

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation Steps

1. Clone the repository:
```bash
git clone https://github.com/analaurafra/bolt_project_custumer.git
cd bolt_project_custumer
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Configure environment variables (if needed):
```bash
cp .env.example .env
```

## 🎯 How to Use

```bash
# Development
npm run dev

# Build for production
npm run build

# Start in production
npm start

# Run tests
npm test
```
